# Poster Theory

An e-commerce platform for curated poster prints — browse collections, customize designs, and order high-quality prints delivered to your door.

## Features

- **Shop by Collection** — Anime, Movies, Music, Minimal, Typography, Abstract, Vintage, Photography, Gaming, Sports, and more
- **Multiple Sizes** — A3, A4, A5, A6, Polaroid, and Pocket formats
- **Layout Options** — Single, Duo, Trio, and Quad panel prints
- **Custom Posters** — Upload your own images and create personalized poster designs
- **User Accounts** — Sign up with email or Google, manage your profile and saved addresses
- **Order Tracking** — View order status from placement to delivery
- **Admin Dashboard** — Manage products, collections, orders, pricing, coupons, and homepage content
- **Secure Payments** — Razorpay integration for safe transactions
